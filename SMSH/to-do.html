Bugs

Features

  • Faster ul/ol with li (.uli & oli)
  • Faster table with tr (row)
  • .. for previous element with same indent
  • Ability to use other SMSH files
  • config.smsh file in smsh directory, accesses this before all other SMSH files (default values by default)
  • Tabs using @ character (like sections)
  • Override default CSS using % character (same as $, but without . (no class))
  • \[string] [replacement] replacement (\a b replaces all instances of \a with b). Longer strings have priority
  • :bg Background color
  • :margin default margin length
  • :theme auto auto detects browser color scheme (when implemented make new default)
  • <: > multi-line comment
  • -c compiler command: Takes an html file as input, takes the body, strips JS from it (incl. onclick and similar), returns smsh with that body. If no -o, replace original file.
  • Ability to override special tags with the compiler. For example, smsh file.smsh :"theme light" -o fileLight.html should compile the file to fileLight.html in light mode, even if the SMSH file specifies dark mode.
